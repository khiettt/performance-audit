## conclusie 

Uit de verschillende tests blijkt dat de website van Klein&Co eigenlijk al best goed presteert. Vooral de laadtijd van de belangrijkste content en de stabiliteit van de pagina zijn sterk. Wel zijn er nog een paar punten die verbeterd kunnen worden, zoals ongebruikte CSS en JavaScript, grote netwerkpayloads en enkele externe bronnen die extra laadtijd veroorzaken. Vooral op mobiel heeft dit meer invloed op de prestaties dan op desktop. Door overbodige code op te ruimen en het aantal externe verzoeken te verminderen, kan de website nog sneller en efficiënter worden.
